算法数据结构      
                         ---- 排序  
实验目的：  
1.熟悉排序的概念。  
2.掌握排序的方法。  
实验内容：  
表结构定义如下  
```c++
typedef struct {
          int key;   //主关键字
           ….    //数据元素其他项的信息可不定义；
         } ElemType;

typedef struct {
          ElemType   *R; //表基址
          int          length;     //表长
}SSTable;
```
基于如上存储结构，设计一个程序,要求在主程序中创建简单的菜单进行选择如下实现如下算法，并在程序中输出每一趟排序的结果以及比较次数和移动次数的计算，并确定该算法的稳定性：  
1．	冒泡排序；  
2．	希尔排序；  
3．	堆排序；  
4．	快速排序；  
5．	选做（归并、基数）   
实验要求：  
1、 程序要添加适当的注释，程序的书写要采用缩进格式。  
2、 程序要具在一定的健壮性，即当输入数据非法时，程序也能适当地做出反应，如插入删除时指定的位置不对等等。  
3、程序要做到界面友好，在程序运行时用户可以根据相应的提示信息进行操作。  
4．在体会中描述如下内容：  
（1）图示按照你的测试数据建立的堆排序、选择排序示意图。  
（2）本次实验过程的体会，实现了几个程序，各个程序的名字是什么？是否自己独立完成？最大的困难是什么？自己准备如何解决这个困难？
